<p><span style="color:#2ecc71;"><strong><span style="font-size:20px;">⭐ ОПИСАНИЕ ПЛАГИНА</span></strong></span></p>

<p style="text-align: justify;"><strong><span style="font-size:16px;">Плагин добавляет систему жизней на ваш сервер Minecraft. При смерти игрок теряет одну жизнь, а при потере всех жизней автоматически банится. Администраторы могут управлять количеством жизней игроков, разбанивать их и устанавливать новое количество жизней.</span></strong></p>

<p style="text-align: justify;"><span style="color:#2ecc71;"><strong><span style="font-size:20px;">🎯 ОСОБЕННОСТИ ПЛАГИНА</span></strong></span></p>

<ul>
	<li><strong><span style="font-size:16px;">&nbsp;💾 Автоматическое сохранение данных в SQLite</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;🌍 Поддержка 7 языков (Русский, English, Українська, Polski, 中文, Fran&ccedil;ais, Espa&ntilde;ol)</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;⚡ Асинхронное кэширование для высокой производительности</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;🎮 Интеграция с PlaceholderAPI</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;⚙️ Гибкая настройка через config.yml</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;🔒 Система прав доступа</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;📊 Настраиваемые команды бана/разбана</span></strong></li>
</ul>

<p>&nbsp;</p>

<p><span style="color:#2ecc71;"><strong><span style="font-size:20px;">📊 PlaceholderAPI</span></strong></span></p>

<ul>
	<li><strong><span style="font-size:16px;">&nbsp;`%lifelimiter_hearts%` - текущее количество жизней игрока</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;`%lifelimiter_max_hearts%` - максимальное количество жизней</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;`%lifelimiter_min_hearts%` - минимальное количество жизней</span></strong></li>
</ul>

<p><strong><span style="color:#2ecc71;"><span style="font-size:20px;">🎮 Команды плагина</span></span></strong></p>

<ul>
	<li><strong><span style="font-size:16px;">&nbsp;`/ll add &lt;игрок&gt; &lt;количество&gt;` - Добавить жизни игроку</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;`/ll give &lt;life-stealer&gt; &lt;игрок&gt;` - Выдать шприц жизни игроку</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;`/ll remove &lt;игрок&gt; &lt;количество&gt;` - Удалить жизни у игрока</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;`/ll unban &lt;игрок&gt; &lt;количество&gt;` - Разбанить игрока и установить количество жизней</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;`/ll reload` - Перезагрузить конфигурацию плагина</span></strong></li>
	<li><strong><span style="font-size:16px;">&nbsp;`/ll help` - Показать справку по командам</span></strong></li>
</ul>
